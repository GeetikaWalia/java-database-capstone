
# Smart Clinic Management System - User Stories

## **Admin User Stories**

### **Epic: Manage Access and System Functionality**
1. **As an admin, I want to create user accounts (doctors, patients), so that they can access the system.**
2. **As an admin, I want to assign roles and permissions, so that users have appropriate access levels.**
3. **As an admin, I want to deactivate or delete user accounts, so that inactive users cannot access the system.**
4. **As an admin, I want to view system usage reports, so that I can monitor performance and user activity.**
5. **As an admin, I want to configure system settings (e.g., appointment limits, working hours), so that the system operates according to clinic policies.**
6. **As an admin, I want to manage doctor schedules, so that patients can book appointments based on availability.**
7. **As an admin, I want to update clinic information (address, contact details), so that patients see accurate details.**

---

## **Patient User Stories**

### **Epic: Book and Manage Appointments**
1. **As a patient, I want to register and create an account, so that I can access appointment booking features.**
2. **As a patient, I want to log in securely, so that my personal health data is protected.**
3. **As a patient, I want to search for doctors by specialty, so that I can find the right doctor for my needs.**
4. **As a patient, I want to view available time slots, so that I can book an appointment conveniently.**
5. **As a patient, I want to book an appointment online, so that I don’t need to visit the clinic physically.**
6. **As a patient, I want to cancel or reschedule my appointment, so that I can manage changes in my schedule.**
7. **As a patient, I want to view my appointment history, so that I can keep track of past visits and prescriptions.**
8. **As a patient, I want to receive notifications/reminders for upcoming appointments, so that I don’t miss them.**

---

## **Doctor User Stories**

### **Epic: Manage Availability and Patient Appointments**
1. **As a doctor, I want to log in securely, so that my professional data is protected.**
2. **As a doctor, I want to set my availability (working hours, days off), so that patients can book appointments accordingly.**
3. **As a doctor, I want to view my appointment schedule, so that I can plan my day effectively.**
4. **As a doctor, I want to update my availability, so that patients see accurate booking options.**
5. **As a doctor, I want to view patient details before an appointment, so that I can prepare for the consultation.**
6. **As a doctor, I want to add prescriptions and notes after an appointment, so that patients have access to their treatment plan.**
